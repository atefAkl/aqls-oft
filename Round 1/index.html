Introduction

  • What's Javascript

    JavaScript is a programming language that enables interactive web pages.

  • Why named JavaScript

    JavaScript was originally developed as a client-side scripting language for web browsers.

  • Javascript Engines

    Some popular JavaScript engines include: V8 (Chrome),
    SpiderMonkey (Firefox),
    Chakra (Edge), and
    SquirrelFish (Safari).

  • How these engines works

    JavaScript engines work by parsing the JavaScript code, converting it into an intermediate representation, and then executing it. They use techniques like Just-In-Time (JIT) compilation to optimize performance.

  • what can JavaScript do in browsers?

    1. Add new elements to html page
    2. Edit styles
    3. Manage interactivity with users
    4. Communicate with servers
    5. Read & write Cookies
    6. Store data locally [localStorage, IndexedDB]

  • what can't JavaScript do in browsers?

    1. Access the file system
    2. Access the network (except through APIs)
    3. Run arbitrary code on the server
    4. Access hardware (e.g., camera, microphone)
    5. Block the main thread

  • why JavaScript is unique?

    1. totally merged with HTML, CSS
    2. Making things easily
    3. Supported with major browsers and run by default

  • Languages OVER JavaScript

    1. CoffeeScript [Sugar Syntax]
    2. TypeScript [Static Typing]
    3. Flow [Static Typing]
    4. Dart [Optimized for UI]
    5. Brython [Python in the Browser]
    6. Kotlin/JS [Kotlin in the Browser]
    7. Elm [Functional Programming]
    8. ReasonML [JavaScript with Type Safety]
    9. Purescript [Strongly Typed Functional Programming]

  • Conclusion

    JavaScript is a versatile language that powers the modern web.

    Its unique features and widespread support make it an essential tool for developers.

    Fully integrated with HTML/CSS.

    There are many languages that get “transpiled” to JavaScript.
